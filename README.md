## ğŸ§© Foundational Principles

### **1. DGCP is a Human-Proof Protocol, not a Data-Collection Protocol.**
- No sensors  
- No automated surveillance  
- No biometric requirements  
- No machine-generated labor inputs  
- No ML scoring or classification  

DGCP respects the worker as the only legitimate origin of Proof.

### **2. Machine Validation must never override Human Interpretation.**
- Machine-Critical fields ensure **technical integrity**  
- Human-Interpretive fields ensure **truth and meaning**  
- Final judgment remains under **Human Governance**, not AI  

This separation creates an ethical, resilient model that scales globally.

## ğŸ”— Minimal Proof Chain

DGCP uses a deliberately minimal chain:

1. **Phone**  
   - Worker captures the real moment of labor  
2. **Metadata**  
   - DMS v1.0.0 defines the minimum fields for Proof integrity  
3. **IPFS CID**  
   - Immutable content addressing  
4. **GitHub SHA256 Registry**  
   - Public, append-only provenance  
5. **Minimal DLT Timestamp**  
   - Global auditability anchor (OTS/Bitcoin)  

This allows DGCP to run even in low-income, low-infrastructure communities.
